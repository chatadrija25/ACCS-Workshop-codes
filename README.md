These are codes used for a workshop conducted by Dr. Pragathi P. Balasubramani, Indian Institute of Technology, Kanpur 
and supported by Annual Conference of Cognitive Science 11(ACCS 11) conducted at the Indian Institute of Technology, Bombay.  

The folders contains 
1) Basic EEG Preprocessing code and add-on files to be used with it. Mainly uses EEGLAB toolbox.
2) GUI interface for GED Generalised Eigen Decomposition (GED).
   You will be able to upload files and play around if you are starting out.
   Detailed GED codes are in the MSR thesis repository.

References are mentioned at the start of each code.
